This demo is a vertical slice meant to demonstrate the game's movement systems as well as how they interact with a basic
version of the combat system.

How to Play

Left/right arrow keys moves the player left/right
Press A to dash
Press Z to Jump
Press X to attack
Press ESC to pause the game
While dashing, you can jump for higher distances

All sprites are placeholders and not final. The zero (player) sprite was was originally created by DoctorAllanGrey on DeviantArt.
Link to the original sprite sheet: https://www.deviantart.com/doctorallangrey/art/Sprites-Megaman-Zero-Mythos-410834488
